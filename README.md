# PowerBI_Restaurant_Project

📖 Overview
This project focuses on analyzing restaurant data using Power BI to uncover valuable insights about cuisines, restaurant distribution, ratings, pricing, and customer preferences. The dashboard transforms raw restaurant data into interactive visualizations that help understand trends and support data-driven decision-making.

🎯 Objectives
Analyze the most popular cuisines.
Identify cities with the highest restaurant concentration.
Explore restaurant ratings and customer preferences.
Study pricing patterns across restaurants.
Create interactive dashboards for better business insights.

🛠️ Tools & Technologies
Power BI
Power Query
DAX
Microsoft Excel / CSV Dataset
Data Visualization

📊 Dashboard Features

Level 1 Analysis
Top Cuisines Analysis
City-wise Restaurant Distribution
Restaurant Ratings Analysis

Level 2 Analysis
Pricing & Cost Analysis
Online Delivery Analysis
Table Booking Analysis

Level 3 Analysis
Geographic Analysis
Restaurant Chains Analysis
Advanced Business Insights

📈 Key Insights
Identified the most popular cuisine categories.
Analyzed restaurant distribution across cities.
Evaluated customer ratings and restaurant performance.
Explored pricing trends and service availability.
Created interactive dashboards for effective decision-making.
